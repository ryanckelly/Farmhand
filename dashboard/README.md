# Stardew Valley Focus Flow Dashboard

An ASCII-style dashboard that visualizes your Stardew Valley progress, financial trends, and session momentum.

## Features

- **Unlocks Tracker**: Community Center bundles, museum donations, cooking recipes, friendships, and skills
- **Financial Analysis**: Current balance, daily averages, trends, and sparkline charts
- **Momentum Analysis**: Hot/cold streaks across 3-session and 7-session windows
  - Detects skill progression bursts
  - Identifies stagnant areas
  - Tracks relationship momentum
  - Shows bundle completion trends

## Quick Start

```bash
# Generate dashboard (creates dashboard.html)
python dashboard_generator.py

# View help
python dashboard_generator.py --help

# Generate with terminal preview
python dashboard_generator.py --preview

# Custom output filename
python dashboard_generator.py --output my_progress.html
```

## Requirements

- Python 3.6+
- Existing save files generated by `session_tracker.py`

## How It Works

The dashboard reads from:
- `save_snapshot.json` - Current game state
- `diary.json` - Session history
- `metrics.json` - Trend data

And generates:
- `dashboard_state.json` - Computed analytics (raw data)
- `dashboard.html` - Visual ASCII dashboard (open in browser)

## What the Dashboard Shows

### Unlocks Progress
Progress bars for major game milestones:
- Community Center bundles (30/31 = 96%)
- Museum donations (66/95 = 69%)
- Cooking recipes, friendships, skills

### Financial Trends
- Current balance
- Daily earning average
- 30-day trend sparkline
- Best earning day

### Momentum Analysis

**3-Session Window** (short-term trends):
- [HOT] High XP gains, strong earnings
- [COLD] No progress, stagnant areas

**7-Session Window** (medium-term patterns):
- [RISING] Leveling up, relationship milestones
- [STALLED] No donations, neglected areas

## Tips

- Run after each gaming session to track progress
- Use momentum insights to identify what needs attention
- Open `dashboard.html` in any web browser
- Works best with at least 7 play sessions recorded

## Troubleshooting

**"Required file not found"**
- Run `python session_tracker.py` first to generate save data

**Terminal preview shows encoding error**
- This is normal on Windows - the HTML file will display correctly
- Open `dashboard.html` in your browser instead

**Momentum shows "Need at least X sessions"**
- Play more sessions and run session_tracker.py after each
- Momentum tracking requires 3+ sessions for short-term, 7+ for long-term

## Implementation Details

- **Zero dependencies**: Pure Python stdlib only
- **Single file**: All code in `dashboard_generator.py`
- **Fast**: Generates in under 1 second
- **Maintainable**: Clear separation between data extraction and rendering
